# In My Closet

## Description

A mobile application designed to help users digitally organize and manage their wardrobe. This app allows users to catalog their clothing items, create outfits, and make mindful fashion choices.

## Installation

npm install

## Features

- Virtual wardrobe organization
- Outfit planning and creation
- Clothing item categorization
- Photo upload capability
- Wardrobe insights and statistics
- User-friendly interface
